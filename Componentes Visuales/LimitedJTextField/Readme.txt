Componente Visual: LimitedJTextField
Descripción del Componente
El LimitedJTextField es un campo de texto personalizado en Java Swing que cambia de color cuando se supera un límite de caracteres definido. Sus características principales son:

Límite de caracteres configurable (valor por defecto: 25)

Cambio automático de colores (fondo y texto) al superar el límite

Selector de colores personalizado para definir los colores de advertencia

Restauración automática a los colores originales cuando el texto vuelve a estar dentro del límite

Mensaje de advertencia al colocar el componente en el diseñador
